# Kontinente Icons

Hier kommen die stilisierten Kontinent-Silhouetten oder Icons rein.

## Benötigte Bilder (6 Stück):

1. **afrika.png** - Afrika (Silhouette oder stilisiertes Icon)
2. **europa.png** - Europa (Silhouette oder stilisiertes Icon)
3. **asien.png** - Asien (Silhouette oder stilisiertes Icon)
4. **nordamerika.png** - Nordamerika (Silhouette oder stilisiertes Icon)
5. **suedamerika.png** - Südamerika (Silhouette oder stilisiertes Icon)
6. **australien.png** - Australien (Silhouette oder stilisiertes Icon)

## Anforderungen:
- Format: PNG (besser für Icons) oder SVG
- Größe: Mindestens 300x300 Pixel
- Stil: Stilisiert, Icon-artig, einfarbig oder mit Akzentfarben
- Hintergrund: Transparent (PNG) oder farbiger Hintergrund
- Lizenz: Creative Commons oder frei nutzbar

## Empfohlene Quellen:
- flaticon.com
- icons8.com
- freepik.com
- thenounproject.com

## Tipps:
- Suche nach "continent silhouette", "continent icon", "africa icon", etc.
- SVG-Dateien sind ideal (skalierbar ohne Qualitätsverlust)
- Alternativ: Farbige Karten-Silhouetten mit den Kontinent-Farben aus der Website
